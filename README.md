
Expense Tracker Data Analysis Project

Objective:
Analyze monthly and category-wise expense data to identify spending patterns and savings opportunities.

Tools Used:
- Python (Pandas, Matplotlib, Seaborn)
- Excel
- Power BI

Project Structure:
- data/: Expense dataset (CSV)
- excel/: Expense dataset (Excel)
- scripts/: Python analysis scripts
- reports/: Insights & recommendations
- powerbi/: Power BI dashboard placeholder

Key Analysis:
- Monthly spending trends
- Category-wise expense distribution
- Savings opportunity identification
